A List of Functions and Arguments

 # bundling functions into a list or dictionary and then calling the functions from the list

# set of functions

def func1(text):
    print(text)

def func2(text):
    print(text)

def func3(text):
    print(text)

def newFunc():
    print('newfunc')

# List of lists holding pointers to each function and the argument. [[functionHere,argument here],[...]]

list1=[[func1,'argument for func1'],[func2,'argument for func2'],[func3,'argument for func3',]]


# access functions through the list of lists [[functionHere,argumentHere],[functionHere,argumentHere]]

print('\nlist calls')
for list in list1: # access the list in the list
    list[0](list[1]) # access each item in the internal list

# Dictionary version: note you can not have duplicate keys in a dict which will be a problem when sequencing function calls

funcDict={func1: "func1",func2: "func2",func3: "func3"}

print(funcDict)

print('dictionary calls here:')
for k,v in funcDict.items():
    k(v)

Code Highlighting: Inserting prism.js into your website

  • Download the prism.js code at prismjs.com
  • Click the download button
  • Select the programming languages you will be writing about
  • Click both download JS and download CSS at the bottom of the page
  • Insert those downloads into your CSS and JS (scripts) folders for your website
  • Link those files inside your HTML file
  • In the Head: <link href="css/prism.css" rel="stylesheet">
  • At the end of the Body: <script src="scripts/prism.js">
  • Specify the language in the HTML in the code tag:
  • <code class="language-python">
